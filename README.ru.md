# Oxide.Ext.MadMappers

[🇬🇧 English](README.md) | 🇷🇺 Русский

Библиотека для расширения и упрощения разработки плагинов для Rust.

---

## 📦 О проекте

Эта библиотека в основном предназначена для оптимизированной разработки плагинов внутри команды Mad Mappers, а также для корректной работы кастомных карт от команды.

При этом другие разработчики также могут использовать часть функционала для лучшей совместимости своих плагинов с работами Mad Mappers.

---

## 📥 Установка

Скачайте актуальную версию DLL:

https://github.com/Mad-Mappers/Oxide.Ext.MadMappers/raw/main/Oxide.Ext.MadMappers.dll

### Шаги:
1. Скачайте файл `.dll`
2. Поместите его в папку `RustDedicated_Data/Managed` на сервере
3. Перезапустите сервер
