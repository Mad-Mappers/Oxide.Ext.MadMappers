# Oxide.Ext.MadMappers

🇬🇧 English | [🇷🇺 Русский](README.ru.md)

A library designed to extend and simplify development for Rust plugins.

---

## 📦 About

This library is primarily intended for optimized plugin development within the Mad Mappers team, as well as ensuring proper functionality of custom maps created by the team.

However, other developers can also use parts of this library to improve compatibility with Mad Mappers content.

---

## 📥 Installation

Download the latest version of the DLL:

https://github.com/Mad-Mappers/Oxide.Ext.MadMappers/raw/main/Oxide.Ext.MadMappers.dll

### Steps:
1. Download the `.dll` file
2. Place it into your server's `RustDedicated_Data/Managed` folder
3. Restart the server
